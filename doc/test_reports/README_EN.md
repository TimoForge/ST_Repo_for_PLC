# Test Reports

[English](./README_EN.md) | [中文](./README_CN.md)

This directory contains all test-related documentation and reports.

### Test Categories
1. Unit Tests
   - Function Block Tests
   - Function Tests
   - Program Tests

2. Integration Tests
   - Module Integration
   - System Integration
   - Communication Tests

3. Performance Tests
   - Response Time
   - Resource Usage
   - Stability Tests

4. Validation Tests
   - Requirement Verification
   - Standard Compliance
   - Safety Validation

### Report Template
```markdown
## Test Report: [Test Name]

### Basic Information
- Test Type:
- Test Date:
- Tester:
- Version Tested:
- Environment:

### Test Scope
Description of test coverage

### Test Cases
| Case ID | Description | Expected Result | Actual Result | Status |
|---------|-------------|-----------------|---------------|--------|
| TC001   | Description | Expected        | Actual        | Pass   |

### Test Results
Detailed test results and analysis

### Issues Found
List of discovered issues

### Recommendations
Improvement suggestions
```
