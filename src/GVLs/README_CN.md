## 📑 目录说明
本目录包含所有全局变量定义，用于PLC程序的全局数据管理。

## 📊 变量分类
### 🔌 硬件接口
- 📥 `GVL_IO` - IO接口变量
  - 🔸 数字输入/输出
  - 🔹 模拟输入/输出
  
### 🏭 工艺变量
- 🎛️ `GVL_Process` - 工艺过程变量
  - 🌡️ 温度控制
  - 💧 流量控制
  - 📊 压力控制
  
### 🛠️ 系统变量
- ⚙️ `GVL_System` - 系统管理变量
  - 🔒 权限管理
  - 📝 日志记录
  - ⚠️ 报警管理

## ⚙️ 开发规范
- 📌 变量分组管理
- 🔍 明确访问权限
- 📋 完整的注释说明
- ⚠️ 变量初始化要求
