# GVLs (全局变量列表)

[English](./README_EN.md) | [中文](./README_CN.md)

本目录包含所有全局变量定义。

### 变量类别
1. 系统变量
   - 系统状态
   - 运行模式
   - 错误代码

2. 设备变量
   - 设备状态
   - 配置参数
   - 运行参数

3. 通信变量
   - 网络参数
   - 通信缓存
   - 协议变量

### 命名规范
1. 前缀规则
   - 系统变量: G_SYS_*
   - 设备变量: G_DEV_*
   - 通信变量: G_COM_*
   
2. 变量属性
   - 常量: G_C_*
   - 配置: G_CFG_*
   - 状态: G_ST_*

### 注意事项
1. 访问控制
   - 明确变量读写权限
   - 合理使用CONSTANT修饰符
   - 必要时使用RETAIN属性

2. 内存管理
   - 注意变量初始化
   - 控制数组大小
   - 合理使用数据类型

3. 文档要求
   - 所有变量必须有注释
   - 说明变量用途和单位
   - 标注安全相关变量
