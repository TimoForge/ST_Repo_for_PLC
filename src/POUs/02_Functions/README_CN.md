## 📑 目录说明
本目录包含可重用的全局函数。

## 🔨 函数分类
### 📐 数学运算
- 🔢 `F_Scale` - 数值缩放
- 📊 `F_Average` - 平均值计算
- 📈 `F_Limit` - 限值处理

### 🔄 数据处理
- 🔍 `F_Search` - 数据查找
- 🔀 `F_Sort` - 数组排序
- 📊 `F_Filter` - 数据过滤

### 🛠️ 辅助功能
- 📝 `F_ToString` - 数值转字符串
- 🔒 `F_CheckAccess` - 权限检查
- ⏱️ `F_TimeFormat` - 时间格式化

## ⚙️ 开发规范
- 📌 命名规则：`F_[功能名称]`
- 🔍 函数必须无副作用
- 📊 参数类型要明确
- ✅ 返回值必须有效
